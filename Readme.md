LTN search crawler
===============
Implement by Scrapy

Usage
-----
```shell
# clone this project from github
git clone https://github.com/LittleYenMin/ltn-news-scrapy.git
# change dir to this project
cd ltn-news-scrapy
# install dependencies
pip3 install -r requirements.txt
# execute 
scrapy runspider crawler.py
```
